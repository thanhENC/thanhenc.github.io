Một đối tượng Object chứa các cặp giá trị string/value không cần thứ tự, được bao trong cặp “{}”, các giá trị bên trong được định dạng “key:value” và chia cách nhau bởi dấu “,”. Value ở đây có thể là chuỗi, số, true- false, null…Có thể xem mô tả cùng ví dụ sau:

Json cũng khá tương đồng với Object trong Javascript. Tuy nhiên, Object trong Json vẫn có những giới hạn như:

  • Key: phải luôn nằm trong dấu ngoặc kép, không được phép là biến số.
  • Value: Chỉ cho phép các kiểu dữ liệu cơ bản: numbers, String, Booleans, arrays, objects, null. Không cho phép function, date, undefined.
  • Không cho phép dấy phẩy cuối cùng như Object trong Javascript.

Các kiểu Object: Kiểu OBJECT, Kiểu OBJECT IN ARRAY, Kiểu NEST OBJECT

Json Array: Mảng trong JSON đại diện cho danh sách thứ tự các giá trị. Mảng trong JSON có thể lưu trữ nhiều giá trị. Nó có thể lưu trữ chuỗi, số, boolean hoặc đối tượng trong mảng JSON.
Trong mảng JSON, các giá trị phải được phân tách bằng dấu phẩy.
[] ( Dấu ngoặc vuông) đại diện cho mảng JSON.
Các loại array (mảng): Mảng JSON của chuỗi, Mảng JSON của các số, Mảng JSON của các giá trị boolean, Mảng JSON của các đối tượng, Mảng đa chiều trong JSON
